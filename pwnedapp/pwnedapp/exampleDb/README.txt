Sample DB insertion code.

How to run:

- install API 'mongodb'
* run 'npm install mongodb' in cmd

- (optional) redirect database to another folder
* create 'mgdb.conf' file
* type into the file: "dbpath = [YOUR_PATH]"
* run 'mongod -f mgdb.conf' in cmd to start database

- run the sample script
* run 'node sample_db_code.js' in cmd




Quick Start:

- browse the database via 'mongo.exe'
